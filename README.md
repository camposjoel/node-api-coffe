# Nodejs - Express API

API using express, middlewares, mongodb, mongoose schemas and authentication with JWT.

## Tools
- nodejs 16
- express
- express-validator
- jsonwebtokens
- mongoose

## Setup

Clone code
```
git clone https://github.com/camposjoel/node-api-coffe.git
```

Install dependencies
```
npm install
```

Run developer mode
```
npm run dev
```
